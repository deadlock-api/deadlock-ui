# Getting Started

Deadlock UI is an open-source Web Components library for [Deadlock](https://store.steampowered.com/app/1422450/Deadlock/) game items. Item data and images are fetched directly from the [Deadlock API](https://assets.deadlock-api.com/) (no local assets or configuration needed).

## Quick Start (CDN)

Add the script tag to your HTML and start using components immediately:

```html
<script type="module" src="https://unpkg.com/@deadlock-api/ui-core/dist/main/main.esm.js"></script>

<dl-item-card class-name="upgrade_clip_size"></dl-item-card>
```
